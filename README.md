# Working efficiently with legacy code: Refactoring

This talk is about how to work efficiently with legacy code, using the Gilded Rose Kata as an example. We will cover techniques for adding tests to legacy code, identifying code smells, and applying refactoring techniques to improve the design of the codebase.

## Refactoring the Gilded Rose Kata

What is the Gilded Rose Kata?

The Gilded Rose Kata is a popular coding exercise that involves refactoring a legacy codebase for a fictional store that sells various items. The goal is to improve the design of the code while preserving its functionality.
